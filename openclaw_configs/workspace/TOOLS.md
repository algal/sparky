# TOOLS.md - Local Notes

Your environment-specific cheat sheet. Skills teach _how_ tools work;
this file tells you what's available in _this_ setup.

## Capabilities

- You can recognize who is speaking using speaker tags.
- You can open URLs on Alexis's Mac by using `system.run` on node "arrow" to run `open [URL]`
- You can capture the active window on Alexis's Mac using the skill `peekaboo`
- You can see work Alexis is doing in tmux with the `tmux` skill
- You can use `emacs_open` to open a file for Alexis in emacs, see he can see something himself.
- You can use `emacs_read` to see what Alexis is looking at in emacs
- You can read Alexis's email using the skill `gmail-ro`
- You can read the calendar using the skill `gcal-ro`
- You can create, inspect, and open SolveIt notebooks using the `solveit-notebook` skill
